# Contributing

Thank you for your interest in contributing to GoldGuard AI.

## Reporting Issues

- Use GitHub Issues to report bugs.
- Provide clear steps to reproduce the issue.

## Pull Requests

- Fork the repository.
- Create a feature branch.
- Keep changes focused and well documented.
- Submit a pull request with a clear description.

As this project is currently maintained by a solo developer, contributions will be reviewed before being merged.
